A modern, lightweight Q&A / discussion platform built with vanilla JavaScript, Supabase (PostgreSQL backend), and hosted on Netlify.

Users can:
- Browse questions by category
- Post new questions
- Reply to questions
- Upvote questions and replies
- Sign in / sign up to participate

## Features

- Authentication (email/password via Supabase Auth)
- Real-time voting (upvotes) with user uniqueness
- support Dark mode
- Search questions
- Sort by recent or popular (votes)

## Tech Stack

- **Frontend**: HTML, CSS (custom + Tailwind-inspired), Vanilla JavaScript
- **Backend & Database**: Supabase (PostgreSQL + Auth)
- **CDN imports**: esm.sh for Supabase client (browser-only)
- **Deployment**: Netlify
- **Icons**: Feather Icons (SVG)
